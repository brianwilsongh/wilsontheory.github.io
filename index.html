About

I'm a results-oriented, language-agnostic software development engineer. My main programming languages are Java and Python. I'm interested in topics like big data processing frameworks, socket programming, and multithreading. Outside of work I'm a dad, a writer, and a bit of a health nut.

I have experience working with cloud technologies (AWS, Google Cloud), databases (MySQL, PostgreSQL, MongoDB), source control software (Github, Bitbucket), automation frameworks (Selenium, HTMLUnit), package managers (maven, npm, pip), web frameworks (Java EE, Spring, Node.js, React.js, Rails) and some other things including mobile development on Android. My most popular mobile app was used in over 50 countries, and I'm excited to make another in the near future.
